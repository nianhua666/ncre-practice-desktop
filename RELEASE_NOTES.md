# Release Notes

## v0.3.3

- 新增“导出复习计划 Markdown”功能
- 新增“导出错题本 Markdown”功能
- 新增后端报告导出接口
- 前端新增一键导出按钮，便于打印、分享和二次整理

## v0.3.2

- 修复打包流程，清理 `build/` 中间目录，避免误点不可运行的中间 EXE
- 明确区分正式可运行产物 `dist\NCREPractice\NCREPractice.exe`
- 完成打包版 EXE 再验证，确认正式产物可启动、可访问接口

## v0.3.1

- 打磨前端概览页，强化专题掌握度和复习建议展示
- 新增 AI 设置页连接测试按钮
- 新增 Sub2API 最新兼容逻辑与自动协议回退
- 新增 GitHub Actions CI 工作流
- 打包脚本自动生成 release zip，并对打包版 EXE 做冒烟测试

## v0.3.0

- 内置多科目题库，重点支持二级 C 语言程序设计
- 二级 C 题库扩展到 420 题，其中高频题 390 题
- 支持系统组卷、专题冲刺、弱项补刷、错题重练
- 支持成绩记录、答题快照、错因分析、专题掌握度、自动复习建议
- 支持 OpenAI Compatible、Sub2API 最新兼容接入与自动协议回退
- 提供 Windows EXE 打包输出、打包版冒烟测试与 release zip 产物
